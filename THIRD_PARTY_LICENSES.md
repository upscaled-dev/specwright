### Why this source

The Cucumber organisation's canonical grammar previously lived at
`cucumber/common`, but that repository has been archived and the grammar
file is no longer reachable at a stable URL. `alexkrechik/VSCucumberAutoComplete`
ships the same canonical multi-language Gherkin grammar (covers all
Gherkin dialects, scenario outlines, doc strings, tags, tables) under
the MIT license, which is compatible with this extension's MIT license.
